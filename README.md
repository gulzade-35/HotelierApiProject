:hotel: Kurumsal Otel Rezervasyon Projesi

:rocket: Proje Hakkında 
RapidAPI üzerinden sağlanan Booking.com API kullanılarak geliştirilmiş modern bir otel listeleme ve detay görüntüleme web uygulamasıdır. Kullanıcıların belirlediği şehre, tarihlere ve kişi sayısına göre uygun otelleri listeleyip detaylarını inceleyebilecekleri bir yapı sunar.
